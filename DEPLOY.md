## How to deploy in 2 minutes:

### Step 1: Create a GitHub repository
1. Go to https://github.com/new
2. Create a new repository called `fandango-sales-database`
3. Initialize with README (optional)
4. Click "Create repository"

### Step 2: Push your files to GitHub
In your terminal (or GitHub Desktop):

```bash
cd /path/to/vercel_deploy
git init
git add .
git commit -m "Initial commit: Sales Database widget"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/fandango-sales-database.git
git push -u origin main
```

(Replace `YOUR_USERNAME` with your GitHub username)

### Step 3: Deploy to Vercel
1. Go to https://vercel.com
2. Sign up or log in with GitHub
3. Click "New Project"
4. Select your `fandango-sales-database` repository
5. Click "Import"
6. Vercel will auto-detect the settings (no need to change anything)
7. Click "Deploy"
8. Wait ~2 minutes for deployment to complete
9. Copy the URL it gives you (e.g., `https://fandango-sales-database.vercel.app`)
